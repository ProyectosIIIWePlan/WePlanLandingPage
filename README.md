#WePlan — Landing Page
Este es el repositorio de la Landing Page de WePlanDemo, una aplicación web diseñada para la gestión de viajes y gastos en grupo. Esta página forma parte de una demo técnica preliminar desarrollada para presentación académica.

A continuación se detallan las características principales y las tecnologías utilizadas.

 Características de la Landing Page
 Presentación visual del proyecto con diseño responsive

 Explicación clara de funcionalidades clave: viaje, bote, gestión de gastos

 Presentación embebida vía Canva

 Formulario de registro para usuarios interesados

 Menú adaptable para dispositivos móviles

 Funcionalidades del Proyecto WePlan
 Viaje
Permite a los usuarios crear planes de viaje. En estos, se seleccionan los billetes de transporte y alojamiento previamente para dividir el precio entre los participantes antes de realizar la compra.

 Bote
Permite seleccionar un importe común que se divide entre los participantes. Una vez recaudado, se puede utilizar mediante una tarjeta virtual que el administrador (y usuarios autorizados) pueden usar para pagar. También incluye una función para devolver el dinero de forma equitativa.
